---

# What is it?
<ul>
<Appear>
    <li>CSS written in JavaScript template literals</li>
    <li>Applied to an element and used as a React component</li>
    <li>Written in the same file as your JS component</li>
</Appear>
</ul>

<Notes>
- Became popular in the early days of atomic design thinking, before component-based architecture on the front-end really took off
- Early example of component thinking in CSS
- Supports features like media queries and animation as well
</Notes>
